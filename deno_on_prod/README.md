# demo for slides

## examples

```bash
node bugsnag/node/main.js
```

```bash
deno bugsnag/deno/main.ts
```

```bash
time node fibo.js 42
```

```bash
time deno fibo.ts 42
```

```bash
rustc fibo.rs
time ./fibo 42
```

## miscellaneous

```bash
rustfmt fibo.rs
```

```bash
deno fmt --ignore=node_modules
```
